# Розпродаж з гаража · Івано-Франківськ

Personal one-page clear-out card (not a shop). Hosted via **GitHub Pages**.

## Live URL (after Pages is on)

https://barf2777.github.io/garage-clearout-if/

## Setup once

1. Fill `CONTACT`, `OLX_LINKS`, and bump `LAST_UPDATED` in `index.html` (bottom `<script>`).
2. Repo **Settings → Pages →** Deploy from branch `main` → folder `/ (root)`.
3. Wait ~1–2 min, open the URL above.

## Edit later

Change `index.html`, set `LAST_UPDATED` to now (ISO, `+03:00`), commit, push — the site updates.
